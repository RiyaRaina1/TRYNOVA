## 📌 Overview

**TRYNOVA** is a modern full-stack luxury streetwear e-commerce platform built with a premium dark-themed UI and enterprise-grade backend architecture.

The platform delivers a complete online shopping experience with product browsing, smart filtering, cart management, secure checkout, authentication, wishlist, and customer dashboard.

It also includes a powerful **Admin Panel** for managing products, users, orders, coupons, analytics, and invoice generation.
